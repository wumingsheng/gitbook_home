# 技巧19:命令行模式

几乎所有的功能都提供了命令行模式

![](/assets/20190510124730.png)

1. 用行号作为地址`n,m`, 例如`1,3p`打印1到3行
2. .代表当前行
3. $代表最后一行, 例如`:0,$p`
4. %代表所有行
5. `'<,'>`代表选择区域


## 缩写

一下缩写没有全部经过验证

  * - t: copy
  * - p: pring
  * - m: move
  * - d: delete
  * - y: yank
  * - p: put
  * - j: join
  * - s: substitude
  * - normal: normal
  * - g: global


> `:'<,'>normal I * `: 给选择区域的每一行行首插入一个星号
> :1,2j多行合并一行

![](/assets/20190510132154.png)


1. `:% normal A;` 每一行末尾添加一个分号
2. `:% normal i//` 每一行行首添加//注释
3. 重复执行上次的命令`:@:`
4. 运行shell命令`:!ls`
5. 将shell命令输入到文件`:r!ls -l`
